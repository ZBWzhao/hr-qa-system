# HR Copilot 前端

## 技术栈
- Vue 3 + Vite
- Element Plus
- Pinia 状态管理
- Vue Router
- Axios
- ECharts

## 启动步骤

```bash
# 1. 安装依赖
npm install

# 2. 启动开发服务器
npm run dev
```

## 访问地址
- 前端页面: http://localhost:3000

## 目录结构
```
src/
├── api/          # API接口封装 (18个)
├── assets/       # 静态资源
├── components/   # 公共组件 (Layout)
├── router/       # 路由配置
├── stores/       # Pinia状态管理
├── utils/        # 工具函数 (Axios封装)
└── views/        # 页面视图 (21个)
```
